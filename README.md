# Akash Tripathi | Full-Stack Developer Portfolio

Welcome to my personal portfolio! This is a modern, interactive web application built with **Next.js 15**, **Tailwind CSS**, and **Framer Motion**, designed to showcase my journey as a Full-Stack Developer and Competitive Programmer.

## 🚀 Features

- **Custom Interactive Cursor**: A bespoke cursor with hover-triggered ring expansions and magnetic interaction.
- **Premium Glassmorphism Design**: High-end UI with subtle gradients, blurs, and border glows.
- **Unified Experience Timeline**: A merged view of professional work and academic journey (B.Tech at IIIT Gwalior).
- **Dynamic Milestones Section**: Visualizing achievements and positions of responsibility in a split-timeline format.
- **Interactive Project Showcases**: Featured projects with detailed tech pills, source code links, and live demos.
- **Micro-interactions**: Magnetic buttons, sleek section transitions, and responsive parallax effects.
- **Contact Integration**: Fully functional contact form integrated with Formspree.io.

## 🛠️ Tech Stack

- **Framework**: Next.js 15 (App Router, Turbopack)
- **Styling**: Tailwind CSS (with CSS Variables & Custom Themes)
- **Animations**: Framer Motion, GSAP
- **Icons**: Lucide React
- **Theming**: Next-Themes (Violet/Dark Theme optimized)
- **Deployment**: Vercel

## 📂 Project Structure

- `src/app`: Next.js App Router pages and layout.
- `src/components/sections`: Interactive page sections (Hero, Experience, Projects, Milestones, etc.).
- `src/components/ui`: Highly reusable UI components (GlowingButton, CustomCursor, TechPill).
- `src/lib/data.ts`: Centralized source of truth for all portfolio content.
- `src/lib/utils.ts`: Utility functions for Tailwind class merging.

## 🏁 Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
